# Project: Config & Settings Centralization

## Overview
This repository serves as a centralized source of truth for my personal configuration files, PowerShell profiles, and development sandbox scripts.

## Directory Structure
- `/powershell`: Contains custom profiles, aliases, and environment configurations.
- `/sandbox`: Contains experimental scripts, snippets, and toolchain configurations.

## Goal
To provide a clean, documented, and reproducible environment setup.
